# Python package build and release checklist

Run commands from the repository root unless noted otherwise.

1. Update the version in `Python/rdlocrand/setup.cfg`.

2. Install build/release tools in the active environment:

   python -m pip install --upgrade build twine

3. Run local package checks:

   python scripts/check-python.py --tests --no-install
   python scripts/profile-python-hotpaths.py --quick

4. Build source and wheel distributions:

   python scripts/check-python.py --build --no-install

5. Verify the built distributions:

   python -m twine check Python/rdlocrand/dist/*

6. Upload to TestPyPI:

   python -m twine upload --repository testpypi Python/rdlocrand/dist/*

7. Install from TestPyPI for a smoke check:

   python -m pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple rdlocrand==X.Y.Z

8. Upload to PyPI:

   python -m twine upload Python/rdlocrand/dist/*

Do not commit `Python/rdlocrand/dist/`, build directories, or egg-info
metadata. They are generated release artifacts.
